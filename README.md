# my_notepad_app

로컬 LLM(Ollama)용 **데스크톱 노트 앱** — AI에게 노트를 조작하는 도구(tool)를 쥐여줍니다.
(1차: 텍스트 작성 도구 → 2차: excalidraw식 그림 그리기 도구 · 완전 로컬·프라이버시 중심)

## 이 저장소 사용법

이 프로젝트는 정적 문서 대신 **질문 가능한 llm-wiki**로 문서화됩니다.
아키텍처·시스템 구성·설치 세팅을 손으로 다 읽는 대신, **AI에게 자연어로 물어보세요.**

- 🧑 **사람**: [`wiki/`](wiki/README.md)를 탐색하세요.
- 🤖 **AI 도구** (Claude Code, Cursor 등): [`llms.txt`](llms.txt)를 읽고 질문에 답하게 하세요.
  - 예: *"이 프로젝트 아키텍처 설명해줘"*, *"설치는 어떻게 해?"*, *"지금 어디까지 됐어?"*
