## TL;DR
- 메인은 **4영역**: 상단 바(NIM 상태·설정 진입) · 좌측 노트 목록 · 중앙 편집기 · 우측 AI 도우미 패널. + **설정 페이지**(별도 화면, API 키 등록).
- 1차 범위 = 텍스트 작성 도구(중앙 편집기 + AI 패널), 2차 = 캔버스(그림) 모드.
- 목업(v2, 프로덕션 수준 · [DESIGN.md](../../DESIGN.md) Quiet Ink 적용 · 텍스트 최소화): [ui-mockup.html](ui-mockup.html).
- DESIGN.md 준수 예: 보라는 AI 전용이라 노트 **선택 표시는 무채색**으로 처리. AI 작성분엔 보라 흔적(ai-mark).

## 화면 영역
1. **상단 바** — 앱 이름, `NIM 모델·연결 상태`(예: `NVIDIA NIM · 연결됨`), 창 컨트롤.
2. **좌측 사이드바** — 노트 검색, `새 노트`, 최근 노트 목록(선택 노트 강조).
3. **중앙 편집기** — 노트 제목·본문. 상단에 **모드 전환**: `노트(1차)` / `캔버스(2차)`.
   - AI가 작성한 텍스트는 강조 표시 + `AI 작성` 뱃지로 구분.
4. **우측 AI 도우미 패널** — 로컬 모델과의 대화. 도구 실행을 칩으로 표시:
   - `텍스트 작성 도구 실행됨`(1차, 활성) / `그림 그리기 도구 · 2차 예정`(비활성).
   - 하단 입력창으로 질문.
5. **하단 상태바** — "노트는 이 기기에 저장", 저장 상태.
6. **설정 페이지** (별도 화면 · 상단 바 톱니 아이콘으로 진입) — 목업 [settings-mockup.html](settings-mockup.html):
   - **AI 연결(NVIDIA NIM)**: API 키 마스킹 입력 + 눈 토글, `키 발급 →` 링크, 모델 선택, 연결 테스트, "이 기기에 암호화 저장" 안내.
   - **일반**: 테마(시스템/라이트/다크). 하단에 저장 버튼.
   - 최초 실행(키 없음) 시 이 화면으로 유도.

## 색/상태 규칙
- **1차(사용 가능)** = 로컬/안전을 뜻하는 teal 계열.
- **2차(예정)** = amber 계열로 "아직 준비 중" 표시.
- AI 관련 액션 = 보라(accent).
